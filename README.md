# 👩‍⚖️ Crimes Against Women in India (2001–2014)

## 📌 Overview
An in-depth data analysis of crimes against women across India over a 14-year period (2001–2014), using government data from data.gov.in. This project uncovers state-wise trends, crime type breakdowns, and year-on-year patterns — with the goal of informing discussions on improving women's safety in India.

## 🎯 Objectives
- Analyze 5.3 million+ reported crime cases across Indian states (2001–2014)
- Identify top states with highest crime rates against women
- Break down crimes by type: Rape, Kidnapping, Dowry Deaths, Domestic Violence, and more
- Visualize year-wise and state-wise trends using charts and graphs

---

## 📊 Key Findings
- **Total Cases Analyzed:** 5,321,610 reported crimes
- **Top States by Crime Rate:** Uttar Pradesh, Andhra Pradesh, West Bengal
- Steady year-on-year increase in reported cases across most states
- Certain crime categories show disproportionately high rates in specific regions

## 📂 Dataset Fields
| Field | Description |
|---|---|
| State/UT | Indian state or union territory |
| District | District within the state |
| Year | Year of reported crime (2001–2014) |
| Rape | Number of rape cases |
| Kidnapping & Abduction | Kidnapping cases |
| Dowry Deaths | Deaths related to dowry |
| Assault on Women | Physical assault cases |
| Domestic Violence | Cases of domestic abuse |
| Importation of Girls | Human trafficking cases |
